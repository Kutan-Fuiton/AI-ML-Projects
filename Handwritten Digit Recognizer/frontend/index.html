Project Information

Overview: This is a simple MNIST digit recognizer. Draw a digit (0-9) on the canvas and click "Predict" to see the model's guess.

How to Use:

  • Draw a number using your mouse.
  • Click "Predict" to see the AI’s prediction.
  • Click "Clear" to reset the canvas.
  • Toggle between dark and light mode using "Toggle Theme".

How it Works:

  • The backend uses a Convolutional Neural Network (CNN) trained on the MNIST dataset.
  • The drawn image is preprocessed: resized to 28x28, thresholded, normalized, and sent to the model.
  • The model predicts the digit and sends it back to the frontend.

Features: Smooth glassy UI, dark/light mode, responsive canvas, and interactive buttons.
